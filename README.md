# Kasparro Agentic FB Analyst
Multi-agent system for diagnosing Facebook Ads performance.



[Image of Multi-Agent System Architecture Diagram]


## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Add data: Ensure `data/synthetic_fb_ads_undergarments.csv` exists.
3. Set API Key: `export OPENAI_API_KEY=sk-...`

## Usage
Run the analysis:
`python run.py "Analyze why ROAS dropped"`
